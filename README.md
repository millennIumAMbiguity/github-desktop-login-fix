# GitHub Desktop → Auto Login Fix
A one-button solution for the issue caused by changing accounts in GitHub Desktop.\
Resets your name and changes the email with a single key.

## Issue
When you switch accounts on GitHub Desktop, your name and mail dont get updated.\
Because of this, I have on several occasions accidentally committed with the wrong name and mail.

## Usage
After changing account, press <kbd>F1</kbd> or <kbd>F2</kbd> and the script will do its thing.

* <kbd>F1</kbd> Fix and use the first mail from the dropdown.
* <kbd>F2</kbd> Fix and use the second mail from the dropdown.
